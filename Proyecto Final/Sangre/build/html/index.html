Bienvenido a la documentacion del Banco de Sangre¶

Sistema de Sangre.

By Roger Urrutia Jose L. Gonzalez M. Ricardo Tejera Leidys Tello Himani Budhrani Uliano Paz Diego Martinez Hernan Dominguez

Concepto General, crear un sistema de donación de sangre en la cual el donante brinde su información y se pueda ver si este es capaz de donarle a alguna de las personas que necesitan donación. Para esto se realiza una evaluación de los tipos de sangre que este puede aceptar.
